# GPU Repair Lab landing

Static technical landing page for a GPU repair service. The page is written in English and covers diagnostics,
repair scope, validation steps, estimated timing, and FAQ content.

## GitHub Pages

1. Upload `index.html`, `styles.css`, `.nojekyll`, `README.md`, and the `assets/` folder to the repository root.
2. In GitHub, open `Settings` -> `Pages`.
3. In `Build and deployment`, choose `Deploy from a branch`.
4. Select the `main` branch and the `/ (root)` folder.
5. After publication, the site will be available at the GitHub Pages URL for the repository.

Contact details and a request form are intentionally not included. Images are stored locally in `assets/images/`,
with Unsplash attribution preserved in the captions.
